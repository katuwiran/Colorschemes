## Utilities

This is a C# project that defines the colorschemes in this repo, and then generates configruation files for the consumption of software, etc.

### Color Definitions
**backgrounds**

`background` - main background

`background edges` - borders, dividers, etc.

`background overlay` - scroll bars, buttons

**text**

`text faded` - comments

`text inactive` - disabled, unfocused

`text hidden` - toggled off, whitespace

**colors**

`magenta` - object names

`plum` - keywords (including built in types)

`purple` - methods (including `get; set;`)

`blue` - fields, properties, operators, semicolon

`cyan` - text

`yellow` - numerals

`red` - errors

`orange` - warnings

`green` - diff modified

**highlights**

`highlight low` - search results, unpressed buttons

`highlight` - selected text, pressed buttons 

`highlight high` - on hover
